# Support

Thanks for using OryCMS.

## Where to Get Help

- Use GitHub Discussions for questions, ideas, and general community conversation.
- Use GitHub Issues for confirmed bugs, feature requests, and structured support cases.

## Before Opening a Request

Please gather the following when possible:

- OryCMS version or commit
- Environment details
- Steps to reproduce
- Error messages, logs, or screenshots

## Best Channel

- `Q&A` or `General` in Discussions: usage questions and open-ended help
- `Help` issue template: setup or troubleshooting issues
- `Bug Report` issue template: broken or unexpected behavior
- `Feature Request` issue template: product improvements and ideas
