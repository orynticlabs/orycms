# Changelog

All notable changes to `@ory-cms/next` will be documented in this file.

## [0.1.0] — 2026-07-15

### Added

- Initial release — extracted from the OryCMS monorepo
- 46 shadcn/ui primitives (accordion, alert, avatar, button, calendar, chart, dialog, form, select, sidebar, table, and more)
- Admin dashboard components: AppShell, AppSidebar, Dashboard, Topbar, InsightsPanel, PageHeader, PlaceholderPage
- Content management UI: OryCMSCollectionContentPage, OryCMSContentForm, OryCMSContentTable, OryCMSDynamicField
- Collection management UI: OryCMSCollectionsAdminPage, OryCMSCollectionSchemaEditor
- Media library UI: OryCMSMediaLibrary
- `useIsMobile` React hook
- Tailwind CSS stylesheet via `@ory-cms/next/styles.css`
